# JBDON - Business Knowledge for Global Leaders

Premium business knowledge website with 154 articles on Analytics, Marketing, Strategy, Economics, Finance, and Leadership.

## Deploy to Render
1. Push to GitHub
2. render.com → New → Static Site
3. Publish directory: `.`
4. Deploy

## Features
- Knowledge Base Sidebar (all 154 articles by category)
- Design 3 Mega Menu (all articles per category)
- Elegant Share buttons
- Premium Navy+Gold theme
- Global search (Ctrl+K)
